# Go FLIF

Go binding to [FLIF library](https://github.com/FLIF-hub/FLIF).

FLIF is [Free Lossless Image Format](http://flif.info/).


# Dependencies

- [FLIF v0.3](https://github.com/FLIF-hub/FLIF/tree/v0.3)


# Setup

```
cd libs/flif
make all install install-dev
```
